# Over View
This is the basic chatting application.

# USED TECHNOLOGY
    FRONTEND:
        HTML,CSS,JAVASCRIPT
    BACKEND:
        NODE.JS,EXRESS.JS,MONGODB,EJS,etc.
# FROM
Here you should mention your's name and the person name,
who you want to send the message.

# MESSAGE
Enter the message and click on send,
The message has been sent to the person.

# EDIT
The edit option is also provided to edit the chat.

# DELETE
The message can also been deleted.
